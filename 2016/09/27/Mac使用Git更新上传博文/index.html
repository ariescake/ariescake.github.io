Mac使用Git更新上传博文

样式、文章

我们维护自己的博客，不外乎修改下样式和更新博文，可以通过以下两种方式来上传博文到Github仓库，方法简易与复杂取决于个人的习惯了。

1.新增.md文件至_posts文件夹

  • _新增.md文件或是黏贴.md文件至_posts文件夹下，在终端使用以下命令即可上传文章：

    $ hexo clean 清除缓存
    $ hexo generate 生成静态页面至public目录
    $ hexo deploy 将.deploy目录部署到服务器

  • 注意：黏贴到_posts文件夹下的.md文件，需包含固定格式的名称、日期、标签，如下：

    title: XXX

    date: 2011-11-11

    tag: Git, Github, Hexo

    2.Git命令添加博文

  • 打开终端输入cd hexo,进入hexo文件夹

  • 输入 hexo new “title”,创建.md文件

  • 在hexo/source/_posts文件夹中可以看到title.md的文件

  • 使用Markdown编辑器将title.md文件修改后，执行以下操作：

    $ hexo clean 清除缓存

    $ hexo generate 生成静态页面至public目录

    $ hexo deploy 将.deploy目录部署到服务器

  • 打开 ariescake.github.io 查看新文章吧
