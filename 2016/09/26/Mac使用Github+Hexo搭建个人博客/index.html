Mac使用Github+Hexo搭建个人博客

1. 准备工具

Git、Github、Node.js、Hexo

2.Git与Github

​ Git是一款免费、开源的的分布式版本控制系统，而Github提供基于Git的版本

​ 托管服务，初学可参考AndroidDeveloper.廖雪峰的官方网站

安装Git

​ 在Mac OS X上安装Git有两种方法：

  • 安装Homebrew,安装操作均在终端命令中执行完成，安装教程可参考Homebrew，网站语言可选简体中文

  • 安装Xcode,Xcode是Apple官方IDE，集成了Git，低版本没有默认安装，需要在设置中下载安装，而笔者使用的MacOS Sierra支持Xcode 8.0，默认安装Git，只需要在终端输入git查看是否安装成功
  • $ git

Git常用命令清单

使用Github

  • 首先注册Github账号,按照一般注册流程进行注册，然后新建仓库，仓库名称为username.github.io

  • 建立好仓库后，为了能够在本地使用Git管理Github上的项目，需要进行一些配置，一般配置SSH key，生成秘钥并把公钥黏贴到Github

  • Mac默认安装了SSH，在终端输入SSH如果出现下图则说明已经安装了SSH

  • 输入 ssh-keygen-t rsa ,即指定rsa算法生成秘钥，连续按三次回车键（不需要输入密码），就会生成两个文件 id_rsa 和 id_rsa.pub ,其中 id_rsa 是秘钥，id_rad.pub 是公钥。这两个文件默认生成目录是 ~/.ssh,要查看需要开启隐藏文件，然后把 id_rad.pub 的内容黏贴到Github上即可授权成功

    $ ssh-keygen-t rsa

    查看秘钥可以使用sublime text 2,拖拽文件夹IDE即可打开复制

    github操作顺序：

    Settings — SSH and GPG keys — New SSH key — 粘贴 — Add SHH key

    生成秘钥时如需更改文件名称，一次回车后输入文件名称，继续回车即可

  • SSH key 添加成功之后，输入 ssh -T git@github.com 进行测试，如果出现以下提示证明添加成功了

4.Hexo

安装Hexo

  • 输入命令

    $ npm install -g hexo-cli

建站

  • 输入命令

    $ hexo init <folder>

    Hexo会在指定文件夹中新建所需要的文件,第一次创建时不需要<>

  • 输入命令

    $ cd <floder>

    进入文件

    $ npm install

    安装依赖包

本地查看

  • 执行命令前确保在文件下，如果不是则输入命令

    $ cd <floder>

  • 输入命令

    $ hexo generate

    自动根据当前目录下文件，生成静态页

  • 输入命令

    $ hexo server

    运行本地服务

  • 打开http://localhost:4000/.就可以看到本地搭建的博客了

5.部署到Github

修改_config.yml设置

  • 在hexo文件找到_config.yml，修改deploy:

  • 输入命令

    $ cd hexo

    查看hexo文件

    $ npm install hexo-deployer-git - -save

    安装插件

    $ hexo clean

    清除缓存

    $ hexo generate

    生成静态页面至public目录

    $ hexo deploy

    将.deploy目录部署到服务器

6.查看username.github.io

> **博客创建结束**
  • 查看username.github.io就是博客部署到服务端的最终形态
  • 服务器的维护可以使用hexo命令或者github客户端
